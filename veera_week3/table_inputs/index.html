Description Input example Notes
type="button" Needs value attribute for button text
type="checkbox" something else Checkbox text comes after input tag
type="color" With value you can set default color.
type="color" is not supported in Internet Explorer and erlier versions or Safari 9.1 and earlier versions.
type="date" type="date" is not supported in Safari or Internet Explorer 11 and erlier versions.
type="email" type="emails" is not supported in UE9 and erlier.
type="range" type="range" defines a control for entering a number whose exact value is not important (like a slider control).
type="password" type="password" Any forms involving sensitive information like passwords should be served over HTTPS.
type="time" type="time" defines a control for entering a time (no time zone).
